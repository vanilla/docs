## /configuration

```http
GET /api/v1/configuration.ext HTTP/1.1
Host: https://yoursite.vanillaforums.com
```

Retrieve forum configuration.

[__Authentication__](../#making-api-calls): required

### Parameters

_This method does not take any parameters._
